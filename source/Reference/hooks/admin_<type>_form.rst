.. _admintypeform:

#################
admin_<type>_form
#################


*****
Usage
*****

Add form elements to a form created via :php:class:`Omeka_Form_Admin`

*********
Arguments
*********

:php:class:`Omeka_Form_Admin` form
    The form object
    
:php:class:`Omeka_Record_AbstractRecord` | null record
    The record being edited, if it is included in the form
