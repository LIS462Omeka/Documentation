################
admin_form_files
################

*****
Usage
*****

Append content to the "Files" part of the item form.

To work with the files themselves, get the ``$item`` from the ``$args`` array and look at ``$item->Files``

*********
Arguments
*********

:php:class:`Item` item
    The item object

:php:class:`Omeka_View` view
    The view object
