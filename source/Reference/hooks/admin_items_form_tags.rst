#####################
admin_items_form_tags
#####################

*****
Usage
*****

Add content to the tags form for an item. 

To work with the tags themselves, get the ``$item`` from the ``$args`` array and do ``$item->getTags()``

*********
Arguments
*********

:php:class:`Item` item
    The item object

:php:class:`Omeka_View` view
    The view object
