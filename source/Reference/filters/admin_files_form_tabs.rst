#####################
admin_files_form_tabs
#####################

*****
Usage
*****

Add to or alter the form sections and tabs on the edit page for a File.

*****
Value
*****

``array`` $tabs
    The array of tabs. The key is the label for the tab, and the value is the HTML to display.
    
*********
Arguments
*********

:php:class:`File` file
    The file being edited
